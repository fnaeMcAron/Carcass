# Carcass

###### Этачо

Проект-каркас для остальных моих игрушек на Юнити

Позже возможно сделаю его полноценным "ядром"



###### Принцип сортировки по слоям:

Layer0 - межсценовые (находятся на уровне игры)

Layer1 - привязаны к сцене но глобальные внутри нее (находятся на уровне сцены)

Layer2 - конкретные объекты в сцене (всякие терминалы и скрипты к ним например, прочие объекты со своей логикой)

Layer3 - интерфейс, HUD, UI

Layer4 - статические объекты (пропы, префабы без логики и подобное)
